 Netflix-Tech-Stack-Overview
 Company
Netflix

  Tech Stack
- Frontend: React
- Backend: Java (Spring Boot, Microservices)
- Database: Cassandra, MySQL
- Cloud: AWS
- Messaging: Apache Kafka

   Why These Tools
- React is used to build fast and interactive user interfaces.
- Java and Spring Boot support scalable and reliable backend services.
- Cassandra handles large-scale distributed data.
- AWS provides global scalability and high availability.
- Kafka enables real-time data processing.

   Summary
Netflix uses this tech stack to ensure high performance, scalability, and reliability for millions of users worldwide.
